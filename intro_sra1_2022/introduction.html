Les exercices

Implémentation de code selon un énoncé assez précis
Exécution de tests unitaires, chaque test rapporte des points
A rendre dans les 5 jours suivants
Si rendu ensuite : perte dégressive de points
Rendre le code produit via github classroom (voir consignes)

Le projet

Dans la continuité des sujets d'exercice :

Développement d'un site de e-commerce "drive" (vente à emporter).
Spring + JPA/Hibernate + JSP/JSTL.

à rendre à la dernière séance (github classroom).
Par groupe de 4 personnes. Groupe imposé.

Consignes pour le projet

Créer un compte Github par personne,
Voir consignes supplémentaires sur FUN (pour classroom).

L'activité sur git doit être représentative :
    • de votre avancement
    • de votre contribution individuelle.
Utilisez maven. La compilation doit être simple.
Doit pouvoir créer un War et le déployer dans Jetty, Tomcat ou autre ...

Fournir un README avec projet (voir consignes)

Prévoir une "démo" du projet (20 minutes max en dernière séance)

Merci !
